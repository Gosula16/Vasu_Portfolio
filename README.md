# Vasu Portfolio 🚀

> A futuristic dark glassmorphism portfolio website for **Gosula Venkata Vasu** — Aspiring Software Engineer specializing in AI & ML.

![Portfolio Preview](https://img.shields.io/badge/Status-Live-brightgreen) ![HTML](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## ✨ Features

- **Dark Glassmorphism UI** — Premium frosted glass design with neon glow effects
- **Particle Background** — Interactive animated particle system with connected lines
- **Typing Animation** — Multi-phrase typing effect in the hero section
- **Scroll Reveal** — Elements animate in as you scroll
- **Custom Cursor** — Glowing cursor with smooth follower effect
- **Responsive Design** — Mobile, tablet, and desktop optimized
- **Animated Progress Bars** — Skill proficiency with smooth fill animations
- **Timeline Experience** — Vertical animated timeline for experience section
- **Contact Form** — Functional form with floating labels and submit feedback
- **Smooth Navigation** — Sticky navbar with active section highlighting
- **Scroll Progress Bar** — Glowing top progress indicator

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (Canvas API, IntersectionObserver)
- Google Fonts (Syne, Space Mono, DM Sans)
